# 🖥 "Card Pokemon" 🎯

## 📑 Descrição

Projeto realizado com HTML, CSS, e JavaScript, com a função de modo escuro e claro.

## ⏱ Processo

- Marcação semâncitca
- Propriedades personalizadas de CSS
- JavaScript
- Flexbox
- Design responsivo
- modo escuro

## ⏲ Função

Ao Clicar no botão do Sol, altera a cor de claro para escuro utilizando Javascript.
